A) Merge-Srot:

    O objetivo do algoritmo mergesort é ordenar um array de números. O primeiro passo do algoritmo é dividir o array em array menores, dividindo por dois a quantidade de elementos dentro do array.
    Exemplo:
            [5, 3, 8, 1]
            /           \
        [5, 3]         [8, 1]
        /    \          /    \
    [5]   [3]      [8]    [1]

    O segundo passo do algoritmo é realizar o merge do array de forma ordenada, tendo como base a divisão feita no primeiro passo.

    Para a primeira etapa do algoritmo o processamento pode ser classificado como o(n), tendo em que é necessário validar cada elemento dos arrays.
    Para a segunda etapa é necessário calcular a quantidade de níveis que um array de n elementos teria ao realizar as divisões. Para essa parte do código o big o terá uma função logarítmica. 
    Dessa forma o calculo final é:

    O(n) × O(log n) 

    Então ficaria como resposta final de O(n log n).

B) Multiplicação de matrizes:
    O objetivo é calcular o produto de duas matrizes quadradas de tamanho n×n. O resultado é uma nova matriz C de tamanho n×n, onde cada elemento C[i][j] é calculado pela soma dos produtos entre uma linha de A e uma coluna de B.


    Para calcular um único elemento da matriz C, é necessário realizar n multiplicações e somas, percorrendo todos os elementos de uma linha de A e uma coluna de B. Portanto, o custo por elemento é O(n).
    Como a matriz resultado C possui n×n = n² elementos no total, o custo total é:

        O(n)          ×        O(n²)         =  O(n³)
